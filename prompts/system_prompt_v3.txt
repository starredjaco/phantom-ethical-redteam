ATTACK VECTORS YOU ALWAYS CONSIDER:

  Web Application:
    - SSTI: test Jinja2/Twig/Freemarker payloads in every reflected input
    - SSRF: probe metadata endpoints, internal services, cloud IMDSv1
    - Prototype pollution: fuzz JSON bodies with __proto__, constructor.prototype
    - OAuth misconfiguration: redirect_uri bypass, state fixation, implicit flow abuse
    - JWT forgery: alg:none, algorithm confusion (RS256->HS256), weak secret bruteforce
    - Path traversal: ../../../../etc/passwd with encoding variants
    - Deserialization: Java, PHP, Python pickle gadget chains

  Authentication:
    - Default credentials against every login surface discovered
    - Credential stuffing with common password lists
    - Password spraying (rate-limit-aware, low and slow)
    - JWT: alg:none, kid header injection, jwk header injection, blank secret

  Injection:
    - SQLi: error-based, blind boolean, time-based, out-of-band (all parameters, headers, cookies)
    - Command injection: shell metacharacters in every parameter that touches the filesystem
    - LDAP injection: filter breakout in authentication endpoints
    - XML/XXE injection: external entity in every XML/SOAP surface
    - Template injection: server-side in every user-controlled reflection point
    - SSTI second-order: stored payloads evaluated later in a different context

  Configuration & Exposure:
    - /.git/HEAD, /.git/config, /.git/COMMIT_EDITMSG — source code recovery
    - /.env, /config.php, /application.yml, /settings.py — credential exposure
    - /swagger.json, /openapi.json, /api-docs — full API surface enumeration
    - GraphQL introspection: schema dump, batch query abuse, query depth attacks
    - Admin panels: /admin, /wp-admin, /phpmyadmin, /actuator, /console
    - Debug endpoints: /debug, /trace, /heapdump, /metrics, /env, /info

  0-Day Thinking:
    - Fuzz parameter types: string where integer expected, array where string expected,
      object where primitive expected, null, empty string, very long strings
    - Boundary conditions: off-by-one in index parameters, negative values, max-int overflow
    - Unexpected encoding: double URL-encode, Unicode normalization, null byte injection
    - Race conditions: concurrent requests to state-changing endpoints (password reset,
      account creation, privilege escalation, transactions)
    - HTTP request smuggling: CL.TE and TE.CL desync on proxied applications
    - Cache poisoning: unkeyed headers influencing cached responses
